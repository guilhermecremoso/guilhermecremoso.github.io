Sobre

Este projeto teve origem na metade de 2017, com a intenção de criar uma central de processamento de dados online, facilitando o acesso e eliminando a dependência de softwares e interfaces complexas. O site reune ferramentas úteis relacionadas à foguetes experimentais, assim como notas de estudo feitas durante o desenvolvimento de cada aplicação.

Nossa missão é unificar informações sobre processamento de dados, eletrônica embarcada e foguetes experimentais, desenvolvidos pela equipe UFABC Rocket Design. Esse site é desenvolvido pelo Departamento de Eletrônica da equipe, e todo seu código fonte é aberto!
